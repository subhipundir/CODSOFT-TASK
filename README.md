TASK 2: Movie Rating Prediction

OVERVIEW:

This project focuses on predicting movie ratings using Machine Learning techniques. The IMDb Movies India dataset was used to analyze different movie attributes such as Genre, Director, and Actor. Data preprocessing techniques, including handling missing values and label encoding, were applied to prepare the dataset for model training. A Random Forest Regressor model was then trained to predict movie ratings, and its performance was evaluated using Mean Squared Error (MSE) and Mean Absolute Error (MAE). This project demonstrates the application of data preprocessing, feature engineering, and machine learning in predictive analytics.

DATASET: 

IMDb Movies India Dataset

TECHNOLOGIES USED:

1. Python
2. Pandas
3. Scikit-learn

 MACHINE LEARNING MODEL:
   
   -Random Forest Regressor

   FEATURES USED:

   -Genre
   
   -Director
   
   -Actor 1
   
 EVALUATION METRICS:
 
 -Mean Squared Error (MSE)
 
 -Mean Absolute Error (MAE)

 PROJECT WORKFLOW:

 1. Data Loading
 2. Data Cleaning
 3. Label Encoding
 4. Train-test split
 5. Model Training
